To run project from localhostYou will need Python and Flask installed on your computerTo install pip, run:      sudo apt install python3-pipTo install Flask, run:      sudo apt install python3-flaskTo install this project's dependecies, run:      pip3 install -r requirements.txtDefine the correct file as the default Flask application:Unix Bash (Linux, Mac, etc.):      export FLASK_APP=application.pyWindows CMD:      set FLASK_APP=application.pyWindows PowerShell:      $env:FLASK_APP = "application.py"Export API Key to API_Key variable       export API_KEY=pk_f379ccad6fe543b888a603fe4e3e3fe7Run Flask and you're good to go!      flask run
